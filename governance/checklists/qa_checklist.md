# Checklist de QA

- [ ] `npm run build`
- [ ] `npm run regression:smoke`
- [ ] `npm run lint`
- [ ] Se lint falhar, baseline antes/depois registrado.
- [ ] Erros em arquivos alterados identificados.
- [ ] Smoke test nao regrediu.
- [ ] Navegacao primaria ainda funciona.
- [ ] Safe area mobile considerada para TopBar, BottomNav e FloatingActionPair.
