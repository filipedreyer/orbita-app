# Contrato - Shell Global

## Canonico

TopBar:

- menu
- marca Olys
- Acesso
- Inbox
- Busca

BottomNav:

- Fazer
- Planejar
- Memoria

FloatingActionPair:

- Idea
- Capturar

## Proibido no Shell

- IA fixa na TopBar.
- Relatorios de IA na TopBar.
- Leitura contextual na TopBar.
- Capturar na TopBar.
- Central na BottomNav.
- Admin na navegacao primaria.
- Marca Orbita como marca vigente.

## Acessibilidade

Acesso, Inbox, Busca, Idea e Capturar devem ser icon-only visualmente e possuir `aria-label`.

## Aceitacoes temporarias

- Menu e Acesso podem apontar temporariamente para Central enquanto nao houver superficies dedicadas.
- Busca pode apontar temporariamente para Memoria enquanto nao houver rota global de busca.
- Marca Olys pode usar fallback visual enquanto asset oficial nao existir.

Essas aceitacoes sao dividas, nao decisoes finais.
